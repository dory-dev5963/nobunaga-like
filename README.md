# 戦国覇道 - Browser Edition

信長の野望風の戦国シミュレーションゲーム（ブラウザ版）

## 概要

ブラウザのみで動作する戦国時代の日本を舞台にしたターン制シミュレーションゲームです。
大名を選択し、内政・軍事・外交を駆使して天下統一を目指します。

## 特徴

- **完全オフライン動作**: インターネット接続不要
- **ブラウザのみで完結**: 追加インストール不要
- **セーブ/ロード機能**: localStorage を使用した進行状況の保存
- **15の大名家**: 織田、武田、上杉、徳川など有名な戦国大名を選択可能
- **戦略性**: 内政による国力強化と軍事による領土拡大のバランスが重要

## 遊び方

### 1. ゲームの開始

1. `index.html` をブラウザで開く
2. 「新規ゲーム」を選択
3. プレイしたい大名を選択
4. 「ゲーム開始」をクリック

### 2. 基本操作

#### 地図の操作
- **国をクリック**: 国の詳細情報を表示
- 自国の場合は兵士募集などのアクションが可能

#### ターン進行
1. 自国を選択して内政を実行
2. 隣接する敵国を攻撃（軍事）
3. 「次のターン」ボタンでターン終了
4. AI大名が行動を実行

### 3. 内政システム

「内政」ボタンをクリックして以下のコマンドを実行できます：

- **農業開発**: 石高を増加させる
- **商業発展**: 金を増加させる
- **治安維持**: 民忠を上げる
- **兵士募集**: 兵数を増やす（金500消費）

各コマンドには武将を割り当てる必要があります。
武将の「政治」パラメータが高いほど成功率が上がります。

### 4. 軍事システム

「軍事」ボタンをクリックして戦争を実行：

1. 攻撃元の国を選択
2. 隣接する敵国を選択
3. 出兵する兵数を入力
4. 指揮官（武将）を選択
5. 「攻撃開始」をクリック

**戦闘の仕組み**:
- 攻撃力 = 兵数 × (1 + 指揮官の戦闘力 / 100)
- 防御力 = 敵兵数 × 1.2（地形補正）
- 攻撃力が防御力を上回れば勝利

### 5. ターンシステム

- 1ターン = 1ヶ月
- 12ターンで1年
- 毎ターン開始時に領地から収入を得られる
  - 金収入 = 各領地の金 × 10%
  - 米収入 = 各領地の石高 × 10%

### 6. 勝利条件

**日本全国統一**: すべての国を支配する

### 7. セーブ/ロード

- **セーブ**: 「セーブ」ボタンをクリック
- **ロード**: タイトル画面で「ゲームをロード」を選択

データは localStorage に保存されます。

## 武将パラメータ

各武将には以下のパラメータがあります：

- **戦闘**: 軍事指揮能力（戦闘時の補正に影響）
- **知略**: 戦略的思考力
- **政治**: 内政能力（内政コマンドの成功率に影響）
- **忠誠**: 主君への忠誠度

## プレイのコツ

1. **初期は内政優先**: まず兵士を募集して軍事力を強化
2. **弱小勢力から攻める**: 兵数の少ない隣国を狙う
3. **武将の能力を活用**: 政治の高い武将で内政、戦闘の高い武将で軍事
4. **資源管理**: 金が不足すると兵士が募集できない
5. **ターン収入**: 領地が増えれば収入も増える
